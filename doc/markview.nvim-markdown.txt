>lua
	--- Static configuration options for various types of block quotes.
	---@class markview.config.markdown.block_quotes.opts
	---
	---@field border? string | string[] Text for the border. Use an array to apply different border *per line*.
	---@field border_hl? string | string[] Highlight group for the border. Use an array to apply different highlights *per line*.
	---

	---@field icon? string Icon to show before the block quote title.
	---@field icon_hl? string Highlight group for the icon.
	---
	---@field preview? string Callout/Alert preview string(shown where `>[!string]` was).
	---@field preview_hl? string Highlight group for the preview.
	---@field preview_line_hl? string Highlight group for the preview line *background*. Uses `line_hl` when not set.
	---
	---@field line_hl? string Highlight group for the line *background*.
	---
	---@field title? boolean Should this callout allow titles(`>[!string] <Title>`)? Disabled by default.

	    info_hl = "MarkviewCodeInfo",
	
	    label_direction = "right",
	    label_hl = nil,
	
	    min_width = 60,
	    pad_amount = 2,
	    pad_char = " ",
	
	    default = {
	        block_hl = "MarkviewCode",
	        pad_hl = "MarkviewCode"
	    },
	
	    ["diff"] = {
	        block_hl = function (_, line)
	            if line:match("^%+") then
	                return "MarkviewPalette4";
	            elseif line:match("^%-") then
	                return "MarkviewPalette1";
	            else
	                return "MarkviewCode";
	            end
	        end,
	        pad_hl = "MarkviewCode"
	    },
	
	    style = "block",
	    sign = true,
	},

Changes how tables are shown.

Demo {img:3}

Ôûï{!IMPORTANT]
Ôûï Table overflow is decided from the rendered natural width, including inline
Ôûï decorations and excluding concealed Markdown syntax.

`overflow = "horizontal"` is the default. When a rendered table is wider than
its window, Markview temporarily disables `wrap` for that window and renders the
full table at its natural width. Use Neovim's horizontal scrolling to inspect the
entire table. The original window-local `wrap` value is restored when preview
rendering is disabled or detached.

`overflow = "raw"` preserves the previous behavior and leaves genuinely wide
tables as raw Markdown.
